## Functionality:  
1.Signup/Signin  
2.Categories  of  Ads  --  Single  Level  categories    
3.Search  Ads  
4.Search  Ads  by  category  
5.Submit  an  Ad

2.Manifest.json 
3.Offline  availability  with service-worker.js 
4.If  vendor  updates  the  assets  or  resources  then  app  gets  updated  on  client  

5.Push  notifications,  your  OLX  app  will  send  context-based  alerts  to  tell  users

# 🛒 OLX PWA Clone ❤️

## 📌 Overview

This project is a **Progressive Web App (PWA)** inspired by OLX, built using **Vanilla JavaScript**. It allows users to post ads, search listings, and communicate with buyers/sellers.

---

## 🚀 Features

### 🔐 Authentication

* User Signup / Signin

### 📦 Ads Management

* Create and submit ads
* View ad details (Name, Description, Model, Year, Images)
* Browse ads by category
* Search ads

### 💬 Communication

* Send messages between buyer and seller

### ⭐ User Experience

* Save ads to view later
* Offline access to saved ads

---

## ⚡ Progressive Web App Features

* 📲 Add to Home Screen
* 📄 Manifest.json support
* 🔄 Service Worker for offline usage
* 🚀 Auto update when new version is available
* 🔔 Push Notifications for messages

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)
* Service Workers
* Web APIs
